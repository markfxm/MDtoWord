# 📄 Markdown 转 Word 专家

### 专为排版与公式优化而生的转换工具

这是一个简单、高效、且注重排版质量的 Markdown 转 Word (.docx) 网页工具。它不仅能将 Markdown 转换为 Word，还专门针对**数学公式兼容性**进行了深度优化，非常适合科研人员、学生和文档工作者使用。

---

## ✨ 核心功能

- **🚀 实时同步预览**：采用分栏设计，左侧输入 Markdown，右侧实时显示排版效果。支持**双向同步滚动**与**句子级高亮提示**，让核对更轻松。
- **🧪 完美公式支持**：
    - **标准模式**：默认导出原生 Word 公式（MathML）。注：受限于文档转换协议，下载的文件在 Word 中可能无法直接编辑公式，建议通过“复制预览”功能手动粘贴以获得最佳的可编辑性。
    - **WPS 兼容模式**：一键将公式转换为高清晰度图片并嵌入 Word。彻底解决在 WPS 或旧版 Word 中打开时公式乱码的问题。
- **📋 快捷复制预览**：支持一键将预览区的富文本（含公式）直接复制并手动粘贴到 Word，保持高度一致的排版。
- **📁 多种输入方式**：支持手动粘贴文本、点击按钮上传 `.md` 文件，或直接将文件**拖拽**至输入框。
- **🎨 精美排版**：预设了符合中文阅读习惯的字体及行间距，支持 Emoji 表情转换，导出的文档无需二次调整即可直接使用。
- **🔒 隐私安全**：所有转换过程均在您的**浏览器本地**完成，您的数据绝不会上传至任何服务器，确保信息绝对私密。

---

## 🛠 如何使用

1. **输入文本**：在左侧区域粘贴 Markdown 内容，或通过“上传 MD”按钮/拖拽方式载入文件。
2. **选择选项**（可选）：
    - **公式转为图片**：如果您使用的是 WPS 或旧版 Word，建议勾选此项以确保公式显示正常。
    - **保留换行**：勾选后，Markdown 中的单次换行将被保留。
3. **实时预览**：在右侧预览区确认排版，鼠标悬停可高亮对应句段。
4. **一键导出/复制**：
    - 点击右上角的 **“下载 Word”** 导出文件。
    - 若导出格式不理想，可尝试点击预览区上方的 **“复制预览”** 后手动粘贴。

## 💻 技术实现

本项目基于现代前端技术栈构建：

- **框架**：Vue 3 + Vite
- **解析**：Marked (自定义扩展增强中文兼容性) + KaTeX (数学公式渲染)
- **导出**：html-docx-js-typescript + file-saver
- **公式处理**：支持 MathML 导出及 html2canvas (用于兼容模式下的公式转图)
- **表情支持**：twemoji (全系统一致的 Emoji 体验)
